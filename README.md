# Smart Home Emergency Detection and Energy Monitoring System using KNN

## Project Overview

The Smart Home Emergency Detection and Energy Monitoring System is an artificial intelligence-based project designed to improve home safety and energy efficiency. The system uses the K-Nearest Neighbors (KNN) machine learning algorithm to analyze environmental and energy-related sensor data and predict different smart home conditions.

The system monitors parameters such as temperature, humidity, motion detection, smoke detection, gas leakage, and energy consumption patterns to identify emergency situations and provide intelligent predictions.

## Objectives

* Develop an AI-based smart home monitoring system.
* Detect emergency conditions such as smoke and gas leakage.
* Use machine learning for smart home state prediction.
* Analyze energy consumption behavior.
* Provide real-time simulation and visualization of predictions.

## Features

* Sensor data analysis and preprocessing.
* KNN-based classification model.
* Emergency detection for hazardous conditions.
* Energy usage monitoring.
* Model accuracy evaluation.
* Confusion matrix visualization.
* Interactive Streamlit interface.

## Machine Learning Algorithm

### K-Nearest Neighbors (KNN)

KNN is a supervised machine learning algorithm used for classification tasks. It predicts the class of new data points by comparing them with the closest existing data points in the dataset.

In this project, KNN is used to classify smart home conditions based on sensor readings and detect possible emergency situations.

## System Workflow

1. Collect smart home sensor data.
2. Preprocess and prepare the dataset.
3. Train the KNN machine learning model.
4. Test the model performance.
5. Generate predictions based on input sensor values.
6. Display results through the user interface.
